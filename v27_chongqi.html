张锡纯论"冲气上冲" — 原书 77 节完整摘录

来源：《医学衷中参西录》第五期第三卷 "77．论冲气上冲之病因病状病脉及治法"

回应 "看看原著中对冲气上逆的论述"

一、张锡纯原书 77 节核心论述（完整摘录）

冲气上冲之病甚多，而医者识其病者甚少。即或能识此病，亦多不能洞悉其病因，而施以相当之治法。
冲者，奇经八脉之一，其脉在胞室之两旁，与任脉相连，为肾脏之辅弼，气化相通。是以肾虚之人，冲气多不能收敛，而有上冲之弊。况冲脉之上系原隶阳明胃府，因冲气上冲，胃府之气亦失其息息下行之常（胃气以息息下行为常），或亦转而上逆，阻塞饮食，不能下行，多化痰涎，因腹中膨闷、哕气、呃逆连连不止，甚则两肋疼胀、头目眩晕。
其脉则弦硬而长，乃肝脉之现象也。盖冲气上冲之证，固由于肾脏之虚，亦多由肝气恣横，素性多怒之人，其肝气之暴发，更助冲胃之气上逆，故脉之现象如此。治此证者，宜以敛冲、镇冲为主，而以降胃、平肝之药佐之。其脉象数而觉热者，宜再辅以滋阴退热之品。

关键提炼：

  • 病机：肾虚不摄 + 肝气恣横 → 冲气上冲 + 胃气上逆
  • 主脉：弦硬而长（这是张锡纯对"冲气上冲"的核心脉象定义）
  • 脉位：肝脉（弦硬）+ 冲脉（长）
  • 治法：敛冲、镇冲 + 降胃、平肝

二、肝肾阴虚 → 冲气上逆（病机链）

张锡纯在第 79 节"论火不归原"中明确论述：

肝肾阴虚，其气化不能固摄，则肝气忿急，可透膈以干大气，肾气膨胀，可挟冲气上冲。则肝气可挟所寄之相火上逆，肾气可挟副肾脏之冲气上逆。于是逆气上干，排挤胸中喉中皆不能容受外气则喘作矣。

病机链条（张锡纯原文）：

  1. 肝肾阴虚
  2. 肝气化不固摄
  3. 肝气忿急 + 肾气膨胀
  4. 肝气挟相火上逆 + 肾气挟冲气上冲
  5. 逆气上干胸中喉中
  6. 排挤大气 → 喘作

关键论断："肝肾阴虚" = 冲气上逆的根源。这跟你的判断完全一致。

三、典型验案 — 冲气上冲兼奔豚

患者：天津张××，年四十五岁
病因：初秋之时，患赤白痢证，医者两次用大黄下之，其痢愈而变为此证。
证候：每夜间当丑寅之交，有气起自下焦挟热上冲，行至中焦觉闷而且热，心中烦乱，迟十数分钟其气上出为呃，热即随之消矣。
脉象：大致近和平，惟两尺稍浮，按之不实
诊断：此因病痢时，连服大黄下之，伤其下焦气化，而下焦之冲遂挟肾中之相火上冲也。其在丑寅之交者，阳气上升之时也。
处方：桂枝加桂汤加减：桂枝尖四钱、生怀山药一两、生芡实六钱、清半夏四钱、生杭芍四钱、生龙骨四钱、生牡蛎四钱、生麦芽三钱、生鸡内金二钱、黄柏二钱、甘草二钱
效果：两剂病愈强半，原方减桂枝加萸肉、枸杞各四钱，三剂全愈

本案的脉诊关键："两尺稍浮，按之不实" = 肾气虚不摄 + 冲气上冲。张锡纯认为"两尺按之不实"是下焦肾虚的指征，是冲气上冲的根本原因。

四、胃气不降验案 — 弦长右部尤甚

患者：大城王××妻，年近四旬
病因：禀性暴烈，处境多不顺
证候：饭后觉食停胃中，似有气上冲阻其下行，大便旬日始下，多日不下时则呕吐
脉象：左右皆弦，右脉弦而且长，重诊颇实，至数照常
诊断：弦为肝脉，弦而且长则冲脉也。弦长之脉，见于右部，尤按之颇实，此又为胃气上逆之脉。肝胃冲三经之气化皆有升无降。
治法：泻肝、降胃、镇冲

本案的脉诊关键：张锡纯明确指出 "弦 = 肝脉，弦长 = 冲脉，弦长见于右部 = 胃气上逆"。三经同病：肝、胃、冲。

五、怒激肝胆冲气上冲验案（升陷汤段）

诊视：其脉左关弦硬，右寸无力，精思良久，恍然悟曰：此必怒激肝胆之火，上冲胃气。
病机：胃气本下行者也，因肝胆之火冲之，转而上逆，并迫肺之吸气。
脉位分析：
  • 左关弦硬 = 肝气暴发
  • 右寸无力 = 肺气虚（大气下陷）
  • 两者结合 = 肝气上冲 + 肺气虚 + 大气下陷 + 冲气上冲

六、降胃镇冲汤 — 张锡纯治冲气上冲主方

主治：冲气上冲兼胃气上逆，致气海元阳随之浮越者。
脉象：脉多弦长有力，右部尤甚，李士材脉诀歌括所谓"直上直下"也
证候：觉胸中满闷烦热，时作呃逆，多吐痰涎，剧者觉痰火与上冲之气杜塞咽喉，几不能息

方剂组成（张锡纯常用）：

生龙骨、生牡蛎、代赭石各八钱，生山药、生芡实各六钱，半夏、生杭芍各四钱，芒硝、苏子各二钱，浓朴、甘草各钱半

方义：

  • 龙骨、牡蛎 = 敛冲、镇冲（主药）
  • 代赭石 = 降胃、镇冲、引相火下行
  • 生山药、生芡实 = 补肾、收敛下焦气化
  • 半夏 = 降逆化痰
  • 生杭芍 = 敛肝、平肝
  • 芒硝 = 通便（兼清热）

八、张锡纯原书的关键洞察（4 点）

① 冲气上冲 ≠ 肝气上逆 ≠ 胃气上逆

三者经常并发，但各有不同：

  • 冲气上冲 = 肾虚 + 冲脉不摄（根在肾）
  • 肝气上逆 = 肝气恣横（根在肝）
  • 胃气上逆 = 胃失和降（根在胃）
  • 但"冲脉上系原隶阳明"，冲气上冲必影响胃气；肝气恣横更助冲胃之气上逆

② 冲气上冲的核心脉象 = "弦硬而长"

张锡纯在 77 节明确说：

其脉则弦硬而长，乃肝脉之现象也。

"弦" = 肝，"硬" = 肝气暴发，"长" = 冲脉上冲。三者合为"冲气上冲"。

③ 冲气上冲常兼肝火/相火

张锡纯用代赭石"引浮越之相火下行"。所以治疗时除了敛冲镇冲，还要兼顾相火。
