# 仿张锡纯辨证施治系统

> 基于《医学衷中参西录》（1974 河北人民出版社修订本）原文构建的中医辨证施治辅助工具

## 项目目标

模仿张锡纯本人临证思路：四诊合参 → 病名/证型诊断 → 方剂推荐 → 方解 → 真实医案参考

## 知识库来源

- PDF：`/workspace/.uploads/f61ca682-5c5e-4ca9-8df7-fbd1348a5069_医学衷中参西录.pdf`（879 页）
- 版本：1974 年第二版，河北人民出版社
- OCR 原文：仅供本项目内部使用，不对外发布

## 核心规则（必须遵守）

1. 知识库 100% 来自《衷中参西录》原文，不杜撰
2. 脉象必须可追溯到真实医案
3. 脉诊分左/右手（左寸/左关/左尺 + 右寸/右关/右尺）
4. 三诊：望+问+切（闻留占位）
5. 静态网页，纯前端
