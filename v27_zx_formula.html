一、19 项张锡纯特色脉象（28 脉之外的"独家词汇"）

张锡纯原书里高频出现的脉象组合，不在 28 脉标准脉名里，是他独有的临床描述：

关前39 案
辨证：上焦虚损 / 大气下陷（升陷汤主方特征脉位）
"关前" = 关脉前段，对应上焦。脉"关前微弱不起"= 大气下陷实证。
常见方剂: 升陷汤 理郁升陷汤 回阳升陷汤
重按不实22 案
辨证：里虚 / 气虚 / 阴虚
"重按不实"= 重按无根，里气已虚。常用于气阴两虚证。
常见方剂: 白虎加人参汤 滋阴清燥汤
按之即无22 案
辨证：真阴亏 / 元气欲脱（重症）
比"重按无"更重，按下去脉就消失。
常见方剂: 来复汤 既济汤
弦长有力19 案
辨证：反教科书：不是肝阳上亢，而是阳明实热！
张锡纯 4 案用白虎加人参汤。
常见方剂: 白虎加人参汤 镇肝熄风汤
重按无18 案
辨证：亡阳 / 气陷 / 肾气绝
"重按无"= 重按无脉，元气欲脱。
常见方剂: 升陷汤 来复汤
按之不实18 案
辨证：里虚 / 阴虚
与"重按不实"略同，但多用于脉诊过程中"按下去不实"。
常见方剂: 滋阴清燥汤 白虎加人参汤
重按无力16 案
辨证：里虚 / 肾虚
"重按无力"= 重按无根。
和缓16 案
辨证：正常 / 胃气尚存
预后好，张锡纯用"和缓"判断疾病向愈。
重按甚实14 案
辨证：里实 / 阳明实热
与"重按不实"相反，= 重按有根，实热内结。
常见方剂: 白虎加人参汤 大承气汤
上盛下虚12 案
辨证：下焦虚损 / 肾不纳气
"上盛下虚"= 上焦邪盛，下焦虚损。喘证常见。
常见方剂: 参赭镇气汤 薯蓣纳气汤
弦硬而长10 案
辨证：肝阳上亢 / 肝风内动
比"弦长"更重，脉管硬 = 内风欲动。
常见方剂: 镇肝熄风汤 建瓴汤
按之有力9 案
辨证：里实 / 邪气盛

辨证：气闭 / 厥证（重症）
