一、张锡纯原书 5 大规律

1 每方剂独立成段（5K-15K 字符）
段结构固定：主治 → 方剂组成 → 方解 → 理论（引用经典）→ 验案（1-N 个）
验案跟在方解后，用"治愈XXX之案, 略登数则"过渡
例：1．升陷汤段 14,699 字符，含 18 案真验案
2 每验案独立，OCR 吃换行
唯一分隔标记 = 新患者叙述开头，不是标点或换行
OCR 把原书换行全删了，必须用 5 大起始模式识别
3 每验案 8 固定元素
严格顺序：患者（身份+年龄+籍贯）→ 病史 → 脉象 → 诊断 → 方剂+加减 → 疗效+剂数 → 按
几乎无例外
4 5 大验案起始模式
① 一人 / 一妇 / 一室女 — 最常见
② 曾治一 / 治一妇 — 既往案例
③ X地X×× — 奉天/西丰/沧州/桓仁 + 姓名 + ××
④ 有兄弟二人 — 多人叙述
⑤ ××来函 — 同行/学生回信（附录）
5 脉象 3 位置 + 5 维度
3 位置触发词：诊其脉 (94%) / 其脉 / 脉象
5 维度定位：总按 / 左手 / 右手 / 关前 / 关后(尺) / 重按 / 轻取
★ 重要 张锡纯不写左寸右寸左关右关左尺右尺！用关前/关后/两尺 替代

四、"濡脉不一定浮" — 17 案实证 用户洞察

现代 TCM 教科书：濡脉 = 浮细而软，主虚证/湿证。但张锡纯 17 案濡脉中 16 案不浮，1 例浮：

原方/位置濡脉原文辨证浮/沉
1．升陷汤 主案 10脉数至六至，兼有沉濡之象大气下陷实证沉
1．升陷汤 主案 10（复诊）已不数，仍然沉濡大气下陷沉
1．大气下陷兼小便不禁左右皆沉濡，而右部寸关之沉濡尤甚大气下陷沉
3．醴泉饮数近六至，而兼有沉濡之象阴虚发热沉
6．治痰点天突穴法左手沉濡痰证沉
6．白术解濡弱，右关尤甚脾胃虚不浮
6．白术解微弱迟濡，脾胃湿寒湿寒不浮
7．太阳病小青龙汤证滑而濡，至数如常外寒内饮不浮
8．肺劳喘咳左部弦硬，右部濡滑，两尺皆重按无根肺肾两虚不浮
11．镇摄汤象甚迟濡冲气上冲不浮
13．血崩证濡弱，而搏近五至气虚血崩不浮
29．少阳三阳合病大而濡，两尺重按颇实下焦实不浮
30．太阴病提纲大而濡脾虚湿盛不浮
51．温病遗方浮而濡，至数如常湿温浮
慢惊风甚迟濡慢惊不浮
升陷汤·周××来函濡弱无力大气下陷不浮
玉液汤微弱迟濡消渴不浮

结论：张锡纯写"濡脉"不拘于"必浮"，按实际脉象记录。濡主虚/湿，可与沉/迟/微/弱/滑/数 任意组合。在升陷汤、醴泉饮、玉液汤等方中沉濡=气陷/阴虚实证。

五、10 字段标准提取结构

case_id : 顺序编号 (1-994) src_file : 来源文件名 src_label : 来源标签 (完整版/子篇) parent_title : 父方剂标题 patient_info { age : 年龄 (一/二/三/... 或数字) gender : 男/女 location : 籍贯 (奉天/西丰/沧州/...) surname : 姓氏 (刘/张/李/...) } pulse_5dim { // 5 维度位置 → 28 脉字符数组 总按 : [] 左手 : [] 右手 : [] 关前 : [] 关后/尺 : [] 重按 : [] 轻取 : [] } pulse_chars : 所有脉象字符去重 [弦,数,洪,...] pulse_signatures : 脉象原文 (最多 5 个) ["其脉沉迟微弱,右部尤甚",...] zx_special : 19 张锡纯特色 [弦长有力,重按不实,关前,...] diagnosis : 诊断原文 "胸中大气下陷" formulas : 张锡纯方剂 (78 方剂库) [升陷汤, ...] modifications : 加减信号 [加人参三钱, 去知母, ...] efficacy : 疗效 [全愈/愈/见轻/死/亡/不起] dose_count : 剂数 [三剂, 数剂, 十剂] has_comment : 是否有按语/或问 comments : 评论原文 [{type:'按', text:'...'}] content : 验案全文 content_length : 字符数
