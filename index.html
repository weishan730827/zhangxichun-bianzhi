4.1 脉诊总论（张锡纯原文·每条带医案）

① 三辨总纲 《衷中参西录》医论·脉论·卷一

张锡纯自述脉诊三步：①辨有力之真假、②辨脉之数与不数、③辨脉之迟数与大气关系。
出处：医论·脉学总论（治大气下陷方·升陷汤方论中详述）

② 真有力 vs 假有力（核心）出处：医方·治阴虚劳热方·醴泉饮方解；医论·论脉之有力无力

"凡脉之真有力者，当于敦浓和缓中见之……皆有洪滑之象。洪者如波涛叠涌，势作起伏；滑者指下滑润，累累如贯珠。"
——《衷中参西录》治阴虚劳热方·醴泉饮方解，PDF p45
脉象辨证左右偏向张锡纯原文 / 医案
洪而有力外感之实热右寸多见（肺） 「外感之热，脉必洪」——医论·论外感热病，PDF p32
滑而有力中焦之热痰右关多见（脾胃） 「痰热壅于中焦，脉必滑」——治痰饮方·期颐饼，PDF p65
弦而有力肝经偏盛（有病之脉）左关多见（肝） 「肝脉弦硬，肝气横恣」——治阴虚劳热方·论肝病治法，PDF p25
弦直大而有力假有力，脾胃真气外泄六部皆可见 「弦硬大而有力，此脾胃真气外泄」——论冲气上冲，PDF p186
敦浓和缓中有力真有力，脾胃健旺右关为主（胃气） 「脉来和缓有神，此胃气充足」——论治泄泻方，PDF p134

医案支撑：沈阳苏××劳嗽案（PDF p46，醴泉饮证）——"脉数六至，虽细弱仍有根蒂" → 治以醴泉饮+赭石+鸡内金，数剂而愈。

③ 数脉的气虚维度 出处：医论·论脉之数；治大气下陷方·升陷汤方论

"世俗医者遇脉数之证，大抵责之阴虚血涸。不知元气虚极莫支者，其脉可至极数。盖人之气血将脱，脉必数也。"
——《衷中参西录》医论·论脉之迟数，PDF p388
数脉 + 左右偏向辨证治法张锡纯原文 / 医案
右寸数 + 右关弱大气下陷兼阴虚升陷汤 + 滋阴 「气分虚而兼阴虚者，当升阳与滋阴并用」——升陷汤方论，PDF p38
左尺数 + 右尺弱肾阴虚、相火妄动滋阴降火 「肾阴虚而相火动者，脉多细数」——医论·论肾，PDF p412
六脉皆数（7-8 至）元气虚极将脱急补元气（山萸肉） 医案：天津张氏妇案（PDF p39）——"脉七至，汗出如洗" → 急用山萸肉二两，汗止脉敛

医案支撑：如治阴虚劳热方·一味薯蓣饮第二案（PDF p47）——"脉甚虚弱，数至七至" → 一味山药四两煮汁当茶，三日愈。

④ 大气下陷脉象总则 出处：医方·治大气下陷方·升陷汤方论

"其脉象沉迟微弱，关前尤甚。其剧者，或六脉不全，或参伍不调。"
——《衷中参西录》治大气下陷方·升陷汤，PDF p32
大气下陷四大特征：关前微弱 / 沉迟 / 微弱或沉濡 / 指下不觉其动

医案支撑（张锡纯原书 9 例升陷汤验案）：
• 案一：一少年，因劳力过度致大气下陷，「脉沉迟微弱，关前尤甚」（PDF p34）
• 案二：胸中满闷，「脉沉迟，右部尤甚」（PDF p34）
• 案三：神昏健忘，「脉沉弱」（PDF p35）
• 案四：气短不足以息，「脉微细欲绝」（PDF p35）
• 案五：寒热往来，「脉弦细而弱」（PDF p35）
• 案六：咽干作渴，「脉数而弱」（PDF p35）
• 案七：怔忡，「脉结代」（PDF p35）
• 案八：满闷，「六脉不全」（PDF p36）
• 案九：参伍不调（PDF p36）

⑤ 左右脉强弱辨证（核心）出处：医方·治大气下陷方·升陷汤方论

"肺之脉诊在右部，故大气下陷，右部之脉多微弱者其常也。然人之元气自肾达肝，自肝达于胸中，为大气之根本。其人或肝肾素虚，或服破肝气之药太过，其左脉或即更形微弱。"
——《衷中参西录》升陷汤方论，PDF p38 —— 强弱比较必须左右对照，不可单看一手
左右脉状态临床含义治法张锡纯原文 / 医案
右手弱、左手正常单纯大气下陷，肝肾未伤升陷汤（最典型） 医案：沈阳苏××，大气下陷案，「右部脉沉迟微弱，左部如常」（PDF p34）
右手弱、左手也弱（同等）大气下陷 + 肝肾元气（大气之根）已伤升陷汤 + 山萸肉 / 培元固本 「大气陷而兼肾虚者，必加山萸肉」——升陷汤方论，PDF p38
右手弱、左手比正常太过（弦长/洪实）大气下陷 + 肝阳上亢 / 冲气上逆升陷汤去升麻，加龙牡镇肝「大气下陷兼肝阳上亢者，去升麻」——升陷汤方论，PDF p38
左手弱、右手正常肝肾阴虚为主，大气未陷不用升陷汤，用左归饮/一贯煎 「肝肾阴虚而左脉弱者，不可升提」——医论·论肝肾虚治法，PDF p412
左手弱、右手太过（弦长）左肝阴虚 + 右冲气/胃气上逆（最常见）镇肝降逆（左补右泻） 医案：治冲气上冲·镇摄汤证，「左关弦细，右关弦长」（PDF p186）
左右皆弦硬肝火 + 冲胃气逆（脑充血之兆）建瓴汤 / 镇肝熄风汤 医案：治内外中风方·建瓴汤证，「左右皆弦硬而数」（PDF p100）
右尺无根 + 左尺细弱命门火衰 + 肾阴亏右归饮 / 肾气丸 「肾中元阳衰惫，右尺必无根」——医论·论肾阳虚，PDF p414

辨证要点：张锡纯绝对强弱看单手断虚实，相对强弱看两手差判病所。例：左关弦硬 + 右关濡弱 = 肝实脾虚；左关弦硬 + 右关也弦硬 = 肝木横逆犯胃。

⑥ 三部九候脏腑定位（左右对照）出处：医论·脉学总论；医方·各方脉象详述

脉位（左）对应（左）脉位（右）对应（右）张锡纯原文
左寸心（心气、心阳）右寸肺（胸中大气，宗气） 「肺脉在右寸，候胸中大气」——升陷汤方论，PDF p32
左关肝胆（肝气、肝火、肝风）右关脾胃（胃气、冲气） 「左关候肝胆，右关候脾胃」——医论·脉论，PDF p388
左尺肾阴（真阴、下焦）右尺命门（肾阳、相火） 「左尺候肾阴，右尺候命门」——医论·脉论，PDF p388

张锡纯最常用辨证模板：左关弦硬（肝阴亏/肝火） + 右关弦长（冲气/胃气上逆）= 镇肝降胃并用。

⑦ 弦硬复合脉（最常见辨证点）出处：治阴虚劳热方·论肝病；医论·论脉之弦硬

脉象 + 左右偏向辨证治则张锡纯原文 / 医案
大而弦，按之似有力非真有力（关部）脾胃真气外泄 + 冲脉上逆镇摄汤（重赭石） 医案：冲气上冲·镇摄汤证，「关部弦硬大」（PDF p186）
弦硬而长，右部尤甚冲气上冲 + 胃气上逆赭石降逆为主 「弦硬而长者，冲气上冲也」——医论·论冲气，PDF p394
左关弦硬肝胆火炽清肝（白芍、玄参） 医案：治肝病·肝火炽盛案，「左关弦硬」（PDF p25）
左右皆弦硬（寸盛尺虚）肝火+冲胃气逆（脑充血之兆）建瓴汤 / 镇肝熄风汤 医案：建瓴汤证，「左右脉皆弦硬而数」（PDF p100）
弦硬而大（尺部重按不实）真气不敛、内风煽动龙骨牡蛎山萸肉 「弦硬大而重按不实者，元气不敛」——医论，PDF p390
弦硬且数（关部）阴分亏损大滋真阴 「弦硬且数，必阴分亏损」——医论·论阴虚，PDF p402

⑧ 特殊脉象（脱证/危证）出处：医方·治阴虚劳热方·一味薯蓣饮验案

脉象辨证治法张锡纯原文 / 医案
脉散乱如水上浮麻，不分至数元气将脱急用山萸肉收敛 医案：一室女温病痰喘，「脉散乱如水波浮麻，不分至数，此将脱之候」（PDF p47）→ 山药两许煮汁饮，喘定脉敛
六脉散乱无根将脱之象重用山萸肉、龙牡 「脉无根者，元气将脱也」——医论·论脱证，PDF p418
脉无起伏为弦，弦而有力紧脉（寒束或大气下陷内有实热）解表或升陷加石膏 「无起伏之弦脉，即紧脉也」——医论·论紧脉，PDF p389
结代脉 / 雀啄气分郁滞（非死证）理气可愈 「结代脉非死脉也，气分郁滞使然」——医论·论结代，PDF p396
