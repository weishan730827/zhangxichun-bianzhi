## 第一节 培脾舒肝汤

### 1.1 主治

> 治因肝气不舒,木郁克土,致脾胃之气不能升降,胸中满闷,常常短气。

### 1.2 方剂组成

| 药物 | 剂量 |
|:---|:---|
| 于术 | 三钱 |
| 生黄耆 | 三钱 |
| 陈皮 | 二钱 |
| 川厚朴 | 二钱 |
| 桂枝尖 | 钱半 |
| 柴胡 | 钱半 |
| 生麦冬 | 二钱 |
| 生杭芍 | 四钱 |
| 生姜 | 二钱 |

### 1.3 方解(张锡纯原书)

> 脾主升清,所以运津液上达;胃主降浊,所以运糟粕下行。白术、黄耆,为补脾胃之正药,同桂枝、柴胡,能助脾气之升,同陈皮、厚朴,能助胃气之降,清升浊降满闷自去,无事专理肝气,而肝气自理。况桂枝、柴胡与麦芽,又皆为舒肝之妙品乎!

> 用芍药者,恐肝气上升,胆火亦随之上升,且以解黄耆、桂枝之热也。用生姜者,取其辛散温通,能浑融肝脾之气化于无间也。

**【方义·张锡纯组方心法】**
- **君药**: 于术三钱、生黄耆三钱——补脾胃,助升清
- **升**: 桂枝尖、柴胡——助脾气上升,兼舒肝
- **降**: 陈皮、川厚朴——助胃气下降
- **舒肝**: 桂枝、柴胡、麦芽(原方所列)
- **佐制**: 生麦冬——养阴润燥
- **佐制**: 生杭芍(重用四钱)——柔肝敛阴,解黄耆桂枝之热,防胆火上升
- **调和**: 生姜——辛散温通,浑融肝脾

### 1.4 临证要义

**【张锡纯·"无事专理肝气"】**:
> 培脾舒肝汤之妙,在"清升浊降满闷自去,无事专理肝气,而肝气自理"。这与升降汤之"少用桂枝、川芎以舒肝气"是一脉相承的——治肝不专治肝,而调中气以治肝。

**【与升陷汤之鉴别】**:
- 升陷汤: 升提大气,治气陷
- 培脾舒肝汤: 调中升降,治肝郁脾弱,气机不运

**【与升降汤之鉴别】**:
- 升降汤: 治肝郁脾弱,以食少胀满为主
- 培脾舒肝汤: 治肝气不舒木郁克土,以胸闷短气为主,术芪更重用,杭芍亦重用

---

## 第二节 培脾舒肝汤与升陷汤系列合用之机

临证时大气下陷与肝郁脾弱常同时出现,可考虑:
- 升陷汤 + 培脾舒肝汤: 治大气下陷兼肝郁脾弱
- 升陷汤 + 升降汤: 治大气下陷兼食少胁满
- 回阳升陷汤 + 培脾舒肝汤: 治大气下陷心肺阳虚兼肝郁
