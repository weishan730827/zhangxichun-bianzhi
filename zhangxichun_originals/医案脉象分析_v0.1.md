## 二、28 脉中张锡纯未使用过的脉象

**2 个 28 脉中的脉象在张锡纯医案中从未出现**: 革, 疾

**原因分析**:
- **革脉**: 在《伤寒论》中主"革脉,妇人则半产漏下,男子则亡血失精",属罕见危急证;张锡纯临床以虚证/阴虚/气陷为主,革脉场景少。
- **疾脉**: 一息七八至(>120次/分),多见于脱证/真阴衰竭;张锡纯用"至数七至/八至"描述,未直接用"疾脉"二字。

## 七、关键发现与启示

### 1. 28 脉使用覆盖率: 26/28 (92.8%)

张锡纯在 264 个医案中**实际使用 26 种 28 脉**,未使用:**革脉、疾脉**。

### 2. 张锡纯核心脉法(超出 28 脉)

| 张锡纯脉法 | 28 脉中无对应 | 张锡纯原话 |
|:---|:---:|:---|
| 真有力 vs 假有力 | ✓ 全新 | "凡脉之真有力者,当于敦浓和缓中见之" |
| 弦硬 | 弦 + 硬 | "弦硬大而有力,此脾胃真气外泄" |
| 脉无根 | ✓ 全新 | "脉无根者,元气将脱也" |
| 和缓 | 缓(≠和缓) | "六部脉中,皆以和缓为贵" |
| 至数描述 | ✓ 全新 | "元气虚极莫支者,其脉可至极数(七至以上)" |

### 3. 张锡纯脉诊次序(实战优先级)

1. **有力/无力**(辨虚实) → 决定补/泻大方向
2. **至数(迟/数)**(辨寒热) → 决定温/清
3. **浮/沉**(辨表里) → 决定解表/治里
4. **真有力 vs 假有力**(辨脾胃/冲气) → 张锡纯独创,防误判
5. **左右脉差**(辨脏腑) → 大气在右寸,肝气在左关
6. **无根/有根**(辨预后) → 元气存亡
