## 八、勾选 + 匹配逻辑

### 8.1 匹配公式

```
score = 0.20 × 诊法匹配
      + 0.20 × 部位匹配
      + 0.25 × 脉象 Jaccard
      + 0.10 × 脏腑匹配
      + 0.10 × 病机匹配
      + 0.15 × 主证 Jaccard
```

### 8.2 匹配规则

**诊法匹配**：
- 输入"按之" → 案例含"按之/重按" = 1.0
- 输入"重按" → 案例含"重按" = 1.0
- 输入"举之" → 案例含"举之/浮取" = 1.0

**部位匹配**（子集 + 上下位）：
- 顶层匹配顶层 = 1.0
- 二级匹配二级 = 1.0
- 顶层匹配二级（如"左部"输入，"左寸"案例）= 0.5
- 二级匹配顶层 = 0.7（张氏习惯把"左寸"也归"左部"）

**脉象 Jaccard**：
- |输入 ∩ 案例| / |输入 ∪ 案例|

**脏腑匹配**：
- 输入"冲脉" → 案例含"冲脉"= 1.0
- 张氏"冲脉上冲"是独立证型

**病机匹配**：
- 输入"阴虚" → 案例含"阴虚"= 1.0
- 包含子串：输入"阴虚" + 案例"肾阴虚" = 1.0

**主证 Jaccard**：
- 同脉象

### 8.3 输出

按 score 排序，Top 10 案例，显示：
- 案例号
- 部位
- 脉象
- 主证
- 病机
- 脏腑
- 诊断
- 方剂
- 张氏原话片段
