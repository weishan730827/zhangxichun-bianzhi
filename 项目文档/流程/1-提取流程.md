# 阶段 1: 提取流程 — 从原书到结构化数据

> tag: v2.7.1 → v2.7.9
> 状态: ✅ 已完成
> 脚本: `docs/extract_v2.6.py`

## 1.1 输入

- **原书**: `zhangxichun_originals/校后_v2_2018完整版_全8期30卷.txt` (1.65 MB, 535 段)
- **段结构**: 医方 164 / 药物 79 / 伤寒 52 / 医论 35 / 医话 44 / 虚劳 10 / 医案 125

## 1.2 提取流程

```
原书 .txt (1.65 MB)
    ↓ 按 `### 篇名 #N X.方名` 切段
535 段结构化
    ↓ 提取 994 验案
cases_v2.7.1.json (2.2 MB)
    ↓
[4 大修复] (v2.7.1)
    ├─ 完整版主源替换 (1689 KB)
    ├─ is_title_line 修复 (### 篇名 #N X.方名 前缀)
    ├─ look-behind 改非捕获组
    └─ 5 维位置细分
    ↓
[3 个核心字段] v2.7.1
    ├─ pulse_chars (基本脉象)
    ├─ pulse_signatures (脉形)
    └─ pulse_5dim (5 维位置)
```

## 1.3 关键修复

| 修复 | 原因 | 效果 |
|---|---|---|
| 完整版主源替换 | 残缺主源漏掉 162 案 | 832 → 994 案 |
| is_title_line 前缀 | 漏 `### 篇名 #N X.方名` | 段结构对齐 |
| look-behind 改非捕获 | 长度不固定报错 | 脉象正则可用 |
| 5 维位置细分 | 张锡纯用 5 种位置词 | 位置精度提升 |
