# 张锡纯脉诊辨证流程研究 — 一比一原著还原

**研究时间**: 2026-07-05
**研究对象**: 《医学衷中参西录》全 8 期 30 卷
**数据规模**: 994 案 + 全部方论原文
**数据源**: 
- `/workspace/zhangxichun-bianzhi/zhangxichun_originals/校后_v2_2018完整版_全8期30卷.txt` (17098 行, 1.7MB)
- `/workspace/zhangxichun-bianzhi/assets/data/cases_unified_v2.7.json` (994 案结构化)

## 核心结论 (一句话)

**张锡纯的脉诊辨证流程不是分 5 步按脉 (重按→轻取→总按→单按→举按)，而是 3 步合一的一句话流程：① 一句话诊脉 (部位+力度+脉象 合并)  →  ② 问因 / 推断病机  →  ③ 投以 / 用 方剂。**

---

## 关键证据

### 证据 1: 994 案中"分步按脉"模式几乎不存在

| 搜索模式 | 出现次数 |
|---------|---------|
| 举按寻 (举按相应) | **0** |
| 轻取重按 / 轻按重按 | **0** |
| 初按重按 | **0** |
| 浮.{0,4}沉 (浮取沉取对比) | 3 (仅连用) |

**结论**: 张氏从未在 994 案中要求医者先重按、再轻取、再总按、再单按。**这 5 步是后人补入的理论，不是张氏原意。**

### 证据 2: 张氏只用一种脉诊格式

| 引出句式 | 出现次数 | 占比 |
|---------|---------|-----|
| 其脉 X | 318 | ~32% |
| 诊其脉 X | 116 | ~12% |
| 切其脉 / 按其脉 / 候其脉 | 0 | 0% |
| **其它 (无明显引出)** | ~560 | ~56% |

张氏**只用两种格式**: "诊其脉 X" 和 "其脉 X"。**永远是一句话描述** (脉象 + 部位 + 力度), 从不分步。

### 证据 3: 994 案中病机推断只有 6 个常用连接词

| 连接词 | 频次 | 作用 |
|-------|-----|------|
| 知其 | 21 | 主观推断 (从脉推病) |
| 因 | 12 | 病因说明 |
| 盖 | 4 | 推论说明 |
| 系 | 50+ | 表归属 |
| 此乃 / 乃 | 13 | 断言 |
| 遂 | 16 | 时间线连接 (随即) |

**关键**: "知其" 是张氏**主观判断**最常用的词 — 他直接说"从脉判断是某病机", 没有"先分析什么, 再分析什么"的分步。

### 证据 4: 994 案单案平均 3.96 个脉 (3-5 个为主)

```
1 种脉:  88 案
2 种脉: 145 案
3 种脉: 171 案
4 种脉: 175 案
5 种脉: 158 案
6 种脉: 115 案
7 种脉:  63 案
```

**结论**: 一个医案的脉诊原话通常包含 3-5 种脉 (浮+数+弦、洪+长+实 等), 是一次性综合诊断, 不是分步单选。

## 张锡纯方论中明示的"脉 → 方"决策规则

这部分最关键 — **张氏在每个方剂的方论里都写明了"什么脉用什么方"**:

| 脉象条件 | 对应方剂 | 原著原话 (方论) |
|---------|---------|----------------|
| 脉洪长 / 洪滑有力 | 白虎汤 | 脉洪滑有力者, 宜用白虎汤 |
| 脉虽洪大而按之不实 | 白虎加人参汤 | 脉虽洪大而按之不实者, 宜用白虎加人参汤 |
| 脉空虚而数按之豁然如无 | 白虎加人参汤 | 脉空虚而数, 按之豁然如无者, 宜白虎加人参汤 |
| 脉细促心中大动 | 白虎加人参汤 | 脉细促, 心中大动… 当用白虎加人参汤 |
| 脉浮而且洪 | 凉解汤/寒解汤 | 脉浮而且洪者, 用拙拟凉解汤, 或寒解汤 |
| 脉洪滑而浮, 苔白浓, 胸烦 | 寒解汤 | 脉洪滑而浮, 舌苔白浓, 胸中烦热者, 宜拙拟寒解汤 |
| 脉沉迟微弱 / 细如丝 / 微弱 | 升陷汤 | **其脉象沉迟微弱, 关前尤甚. 其剧者, 或六脉不全, 或参伍不调.** |
| 脉浮弦 / 脉弦而有力 | 活络效灵丹 / 金铃泻肝汤 | 脉弦而有力, 知系肝气胆火恣盛也 |
| 脉虚而无力或兼数 | 滋阴清燥汤 | 脉虚而无力或兼数者, 宜拙拟滋阴清燥汤 |
| 脉搏异常迟缓 | 理饮汤 | 脉搏异常迟缓者, 宜治以拙拟理饮汤 |
| 脉弦硬而大 | 建瓴汤/镇肝熄风汤 | 脉弦硬而大, 治以建瓴汤 |
| 脉洪大甚实 | 大承气汤 | 脉洪大甚实者, 可用大承气汤下之 |
| 脉实而渴 | 白虎加人参汤 | 脉实而渴者, 宜用白虎加人参汤 |
| 脉虚数 | 资生汤 | 脉甚虚数, 投以资生汤 |
| 脉数至八至 | 六味地黄丸 | 脉数至八至, 先用六味地黄丸 |
| 脉象无力 | 小青龙汤去麻黄 | 脉象无力者, 宜小青龙汤去麻黄 |
| 脉实而至数不数 | 小青龙汤原方加石膏 | 脉实而至数不数者, 宜小青龙汤原方加生石膏 |
| 脉微弱, 此无阳 | 桂枝二越婢一汤 | 脉微弱者, 此无阳也, 不可发汗, 宜桂枝二越婢一汤 |
| 脉结代 | 炙甘草汤 | 脉结代者, 用炙甘草汤 |

## 5 个完整的"脉 → 机 → 方"经典案例

### 案例 1: 案 39 · 滋阴清燥汤
- **脉**: 诊其脉, 数至七等, 微弱欲无
- **机**: 系久病至此, 不可为侨
- **方**: 投以滋阴清燥汤

### 案例 2: 案 328 · 升陷汤
- **脉**: 诊其脉象和平, 重按微弦无力
- **机**: 知其素勤稼穑, 因感冒懒食, 犹枵腹力作, 以致如此
- **方**: 投以升陷汤

### 案例 3: 案 342 · 升陷汤
- **脉**: 诊其脉, 微弱异常
- **机**: 知其胸中大气下陷甚剧
- **方**: 投以升陷汤

### 案例 4: 案 525 · 温降汤
- **脉**: 诊其脉甚和平, 询其心中不觉凉热
- **机**: 知其胃气因寒不降, 转迫血上逆而为衄也
- **方**: 投以拙拟温降汤

### 案例 5: 案 619 · 理郁升陷汤
- **脉**: 诊其脉, 细小而沉
- **机**: 知其小便时努力过甚
- **方**: 治以理郁升陷汤

**共同点**: 都是一句话脉诊 → 一句话病机推断 → 投以方。**没有任何分步按脉。**
